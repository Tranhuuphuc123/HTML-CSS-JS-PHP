-> nguyên nhân là xung đột các bản boostrap
khiến cho không install đc 
>>> hướng giải quyêt

bước 01: kiêm tra bản bootstrap của project
 gõ:  
           npm list bootstrap

bước 02: tiến hành update bootstrap
 gõ:

          npm install bootstrap@latest


bước 03: tiến hành chạy lại lệnh install admin lte 
 gõ:

               npm install admin-lte@3.1.0


bước 04: trong quá trình cài đặt vẫn sẽ phát xin một xung đột nhỏ có thể
chạy lệnh sau để fix các lỗi trên

 gõ:

         npm audit fix
